# MilkRun

MilkRun is a simple webpage used for generating booking links for the Alaska Milk Run and United Island Hopper flights.

Manually entering each segment on the respective airline's website for booking multiple times is a tedious process, especially when researching different flight paths across different days. This website automates a portion of the process by generating links based on a single date (which is typically the desired booking for such flights) and predefined routes. 

Currently, only the Alaska Milk Run is supported. United Island Hopper support is coming in future updates. 

# Features
- Select a single date to populate all date fields
- Predefined routes 
- Save links in persistent browser storage
